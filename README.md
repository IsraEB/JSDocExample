# Requerimientos

-   NodeJS
-   NPM

# Instalación

```console
# sudo npm i -g jsdoc
```

# Uso

Generar la documentación de un archivo

```console
# jsdoc archivo.js
```

# Templates

## BetterDocs

```console
# npm install --save-dev better-docs
# jsdoc archivo.js -t ./node_modules/better-docs
```
